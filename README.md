## Features

- **Network Support**
  - Dualstack: `IPv4/IPv6`
  - Forwarder: `TCP/UDP`
  - Ping Echo: `ICMP`
- **Platform Support**
  - Linux
  - MacOS
  - Windows
  - FreeBSD
  - OpenBSD
- **Proxy Protocol**
  - HTTP
  - Socks4
  - Socks5
  - Shadowsocks
- **Extra Feature**
  - Improved stability without CGO
  - Optimized UDP transmission for game
  - Performed with >2.5Gbps throughput
  - TCP/IP stack powered by **[gVisor](https://github.com/google/gvisor)**
